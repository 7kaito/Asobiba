# Asobiba
学内SNSは大学生しか入れないため、外部の方に知ってもらうために写真を載せておきます。
アプリを作るにあたってサイトやアプリを100〜200個調べ、デザインや機能を徹底的に調べ、なぜこの機能がついているのか、言葉で説明できるようになりました。
このアプリについている機能やデザインには全て意味があり、ユーザーの操作性やデザインにこだわっています。
